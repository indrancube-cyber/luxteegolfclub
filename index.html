Image Optimisation Reflection

To identify poorly optimised images, I would compare file sizes, image dimensions and loading speed. Images that are too large can slow down a website, while images with incorrect dimensions may appear stretched or pixelated. I would use image editing software to resize images to the correct dimensions, choose an appropriate file format such as JPG, PNG or GIF, and apply compression to reduce file size without noticeably affecting quality. I would also preview the webpage on different devices to ensure that all images display correctly and load efficiently. These optimisation techniques improve website performance and create a better user experience.
